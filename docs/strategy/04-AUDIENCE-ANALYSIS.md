## Audience Journey Map

```
AWARENESS           CONSIDERATION         DECISION              ADVOCACY
─────────────────────────────────────────────────────────────────────────
Facebook Reel  →    Visit page       →    WhatsApp inquiry  →   Share post
```

## Key Insights for Website Design

1. **Mobile-first**: >90% of Egyptian Facebook users access via mobile
2. **Arabic-first**: All high-performing content is in Egyptian Arabic
3. **Visual proof**: Construction photos/videos build trust — the website must showcase these prominently
4. **WhatsApp integration**: This is the primary conversion channel — must be prominent on every page
5. **Payment calculator**: Pricing is the #1 inquiry topic — an interactive calculator would reduce friction
6. **Speed**: Target audience may have slower mobile connections — performance is critical
